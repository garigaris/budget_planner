# Отчёт по проекту WebAssembly Budget Planner

## 1. Цель работы

Цель работы: изучить готовый проект WebAssembly, установить компилятор Emscripten, добиться компиляции и запуска приложения, добавить собственное улучшение и закоммитить результат в GitHub-репозиторий.

## 2. Изучение проекта

Проект представляет собой веб-приложение Budget Planner / Expense Tracker. Приложение позволяет добавлять расходы, удалять отдельные записи, очищать список расходов, считать общую сумму расходов и суммы по категориям.

Основные файлы проекта:

- `main.c` — основная логика приложения на языке C;
- `index.html` — HTML-интерфейс приложения;
- `app.js` — JavaScript-код для связи интерфейса с WebAssembly-модулем.

Файл `main.c` содержит структуры для хранения расходов и итогов по категориям. В нём реализованы функции добавления, удаления, очистки расходов и пересчёта итогов. Файл `app.js` обрабатывает форму, вызывает экспортированные функции WebAssembly и обновляет таблицу расходов на странице.

